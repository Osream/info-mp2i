# Fondamentaux des deux TP : Bases de données

*   Définir les principes du paradigme de programmation logique.
*   Maîtriser le vocabulaire des bases de données relationnelles : relation, attribut, domaine, enregistrement, clé primaire, clé étrangère, schéma relationnel.
*   Écrire des requêtes SQL utilisant les éléments suivants :
    *   sélection simple avec `SELECT, FROM, WHERE, DISTINCT, AS` ;
    *   opérateurs élémentaires sur les types `TEXT, INTEGER, REAL `;
    *   opérateurs de comparaison, y compris `IS NULL, IS NOT NULL, LIKE `;
    *   opérateurs booléens `AND, OR, NOT `;
    *   formatage avec `ORDER BY (ASC / DESC), LIMIT, OFFSET `;
    *   opérateurs ensemblistes `UNION, INTERSECT, EXCEPT `;
    *   produit cartésien ;
    *   groupes et filtrage des groupes avec `GROUP BY, HAVING `;
    *   fonctions d’agrégation `MIN, MAX, SUM, AVG, COUNT`, avec ou sans groupe ;
    *   jointures internes (`JOIN`) et externes (`LEFT JOIN`) ;
    *   requêtes imbriquées.
