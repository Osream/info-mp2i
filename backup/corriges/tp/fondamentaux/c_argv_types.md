# Fondamentaux du TP : Ligne de commande et déclaration de types en C

* Écrire une fonction principale de prototype `int main(int argc, char* argv[])`, avec code d'erreur renvoyé si les arguments ne sont pas ceux attendus, et exécuter un programme qui attend des arguments.
* Utiliser les fonctions `atoi` et `atof`.
* Définir un alias avec `typedef` (placé dans le fichier d'entête pour s'en servir depuis d'autres fichiers).
* Définir un type énuméré avec `enum` (éventuellement suivi d'un `typedef`).
* Définir un type structuré avec `struct` (éventuellement suivi d'un `typedef`).
* Connaître toutes les syntaxes pour déclarer une variable avec un type structuré.
* Récupérer (et modifier) la valeur d'un champ d'une structure (avec un `.`) et d'un pointeur vers une structure (avec une `->`).
* Écrire des fonctions manipulant des types déclarés manuellement (y compris avec des pointeurs).
* Distinguer passage par valeur et passage par référence des paramètres.
