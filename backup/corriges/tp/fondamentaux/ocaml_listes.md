# Fondamentaux du TP : Listes OCaml

* Utiliser les syntaxes `[element1 ; element2 ; ...]` et `element1 :: element2 :: ... :: []` pour créer une liste.
* Écrire une fonction récursive avec un filtrage pour parcourir une liste.
* Donner le type et utiliser les fonctions existantes sur les listes : `List.length, List.mem, List.map, List.filter, List.exists, List.for_all, @`.
* Implémenter toutes les fonctions existantes sur les listes avec une fonction récursive avec filtrage.
