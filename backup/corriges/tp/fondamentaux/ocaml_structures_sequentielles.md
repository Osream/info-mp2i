# Fondamentaux du TP : Structures séquentielles en OCaml

* Effectuer une compilation simple avec `ocamlc` ou `ocamlopt`, et une compilation séparée (avec l'option `-c` pour chaque fichier puis `-o` pour la création du fichier exécutable).
* Créer un fichier d'interface `.mli`.
* Utiliser `Sys.argv` pour récupérer les arguments, et `let _ = ...` pour préciser ce qui doit être lancé automatiquement à l'exécution.
* Connaître le type et savoir utiliser les fonctions sur les piles et les files existantes en OCaml : `create, is_empty, push, pop`.
* Connaître le type et savoir utiliser les fonctions sur les tableaux associatifs existantes en OCaml : `create, add, mem, remove, find, find_opt, iter`.
* Ré-écrire les fonctions sur les piles et les files avec les diverses implémentations vues en cours (par un tableau, par maillons chaînés, ...) à l'aide de quelques indications.
