# Fondamentaux du TP : Chaînes de caractères en C

* Savoir qu'une chaîne de caractères est un tableau de caractères contenant une sentinelle (caractère nul `\0`).
* Créer une chaîne de caractères initialisée avec les guillemets (ou les accolades comme les tableaux statiques).
* Déterminer si une chaîne de caractères est mutable (oui si créée avec `char c[]`, non si créée avec `char* c`).
* Accéder à et modifier un caractère d'une chaîne depuis son indice.
* Parcourir une chaîne (grâce à une boucle qui s'arrête quand on croise la sentinelle).
* Écrire une fonction qui prend en paramètre une chaîne et/ou renvoie une chaîne.
* Utiliser les fonctions suivantes sur les chaînes : `strlen, strcpy, strcat, strcmp`.
* Ré-écrire ces quatre fonctions sans utiliser la librairie `string.h`.
* Manipuler des tableaux contenant des chaînes de caractères.
