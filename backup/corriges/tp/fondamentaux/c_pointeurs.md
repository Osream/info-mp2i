# Fondamentaux du TP : Pointeurs et gestion de la mémoire en C

* Utiliser la fonction `scanf` pour réaliser une lecture, avec les formatages `%d, %lf, %c`.
* Connaître le type d'un pointeur en C (si l'objet pointé est de type `t`, le pointeur est de type `t*`).
* Récupérer l'adresse d'une variable existante avec l'opérateur `&`.
* Récupérer (et modifier) la valeur pointée par un pointeur avec l'opérateur `*`.
* Effectuer une allocation dynamique (`malloc`, `sizeof`).
* Libérer une zone allouée dynamiquement (`free`).
* Utiliser les pointeurs pour qu'une fonction ait un effet de bord (modification permanente d'une valeur extérieure à la fonction).
* Utiliser les pointeurs pour écrire une fonction qui doit renvoyer plusieurs résultats.
* Connaître les erreurs liées à la manipulation de mémoire (notamment les erreurs de segmentation et fuites de mémoire).
