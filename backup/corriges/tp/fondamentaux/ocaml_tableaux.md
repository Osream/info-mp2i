# Fondamentaux du TP : Tableaux OCaml

* Ré-écrire (en utilisant uniquement `length` et `make`) les fonctions du module `Array` : `make_matrix, copy, mem, init, exists, for_all, map, iter`.
* Manipuler des tableaux à plusieurs dimensions : création, accès et modification d'un élément, récupération des dimensions.
* Parcourir un tableau multidimensionnel à l'aide de boucles imbriquées.
* Utiliser les fonctions du module `Array` appropriées pour avoir un code plus concis.
