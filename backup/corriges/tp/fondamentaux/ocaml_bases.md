# Fondamentaux du TP : Bases d'OCaml

* Être à l'aise avec le fonctionnement général du toplevel.
* Manipuler les types de base (`int, float, bool, char, string`) et leurs opérateurs.
* Construire et déconstruire un tuple, donner le type d'un tuple.
* Définir une variable locale et globale.
* Écrire une instruction conditionnelle.
* Écrire un filtrage.
* Définir une fonction, y compris avec les mots-clefs `fun` et `function`.
* Définir une fonction récursive, ou plusieurs fonctions récursives mutuelles.
* Donner le type d'une fonction.
* Distinguer fonction curryfiée et décurryfiée.
* Implémenter l'algorithme d'Euclide.
* Implémenter l'algorithme d'exponentiation rapide.
* Respecter les bonnes pratiques de programmation.
