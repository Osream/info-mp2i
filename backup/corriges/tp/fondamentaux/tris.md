# Fondamentaux du TP : Algorithmes de tris

* Implémenter le tri par sélection sur un tableau ou sur une liste, dans les deux langages.
* Implémenter le tri par insertion sur un tableau ou sur une liste, dans les deux langages.
* Implémenter le tri à bulles sur un tableau ou sur une liste, dans les deux langages.
* Analyser ces trois algorithmes de tri (terminaison, correction, complexité).
* Implémenter l'algorithme de recherche dichotomique dans un tableau trié (dans les deux langages), et analyser cet algorithme (terminaison, correction, complexité).
* Reconnaître un problème pour lequel trier les données est pertinent et utiliser un algorithme de tri pour le résoudre.
