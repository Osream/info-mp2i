# Fondamentaux du TP : Bases du C

* Manipuler les types de base (`int, double, bool, char`) et leurs opérateurs.
* Déclarer une variable, affecter une valeur à une variable, y compris avec une affectation composée.
* Définir une constante (`const`).
* Convertir explicitement un type de base vers un autre type de base.
* Écrire une instruction conditionnelle.
* Écrire une boucle non bornée.
* Écrire une boucle bornée.
