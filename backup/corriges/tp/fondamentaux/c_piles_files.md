# Fondamentaux du TP : Implémentation de piles et files en C

* Implémenter *rapidement (5mn max)* une pile par maillons chaînés (avec pointeur vers le sommet).
* Implémenter *rapidement (10mn max)* une file par maillons chaînés (avec pointeurs vers le premier et dernier maillon).
* Calculer la complexité des opérations avec ces implémentations.
* Savoir expliquer (par exemple avec des schémas) d'autres manières d'implémenter une pile ou une file (notamment avec un tableau).
