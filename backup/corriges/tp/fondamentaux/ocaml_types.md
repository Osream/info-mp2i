# Fondamentaux du TP : Déclaration de types en OCaml

* Définir un alias, utiliser les annotations de type dans une fonction pour forcer l'utilisation d'un alias.
* Définir un type produit (éventuellement avec certains champs mutables), déclarer une variable de ce type, récupérer (et modifier) la valeur d'un champ.
* Définir un type somme, déclarer une variable de ce type, manipuler ce type à l'aide de filtrages.
* Écrire des fonctions manipulant des types sommes ou produits, y compris polymorphes ou récursifs.
* Connaître la définition et savoir utiliser le type prédéfini `'a option`.
