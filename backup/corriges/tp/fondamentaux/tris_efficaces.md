# Fondamentaux du TP : Algorithmes de tris efficaces

* Implémenter le tri fusion sur un tableau ou sur une liste, dans les deux langages.
* Implémenter le tri rapide sur une liste, dans les deux langages.
* Implémenter le tri rapide avec partition de Lomuto sur un tableau, dans les deux langages.
* Calculer la complexité d'un algorithme de type « diviser pour régner », et en particulier des tris fusion et rapide.
