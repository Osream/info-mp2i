# Fondamentaux du TP : Logique propositionnelle

* Donner une définition inductive non ambiguë de l’ensemble des formules propositionnelles et définir le type nécessaire à leur implémentation.
* Définir inductivement une fonction sur l’ensemble des formules propositionnelles (taille, hauteur, substitution, ensembles des sous-formules) et écrire la fonction récursive correspondante.
* Écrire une fonction qui détermine si une valuation est un modèle d’une formule propositionnelle.
* Implémenter l’algorithme de Quine :
    * écrire une fonction qui réalise les règles de simplification de Quine ;
    * écrire une fonction qui construit l’arbre de Quine ;
    * écrire une fonction qui utilise l’arbre de Quine pour déterminer si une formule est satisfiable, tautologique ou antilogique.

* Maîtriser le vocabulaire de la logique propositionnelle : variable propositionnelle, arbre syntaxique d’une formule, taille, hauteur, substitution, sous-formule, valuation, modèle, conséquence sémantique, équivalence sémantique, satisfiabilité, antilogie, tautologie, problème SAT, quantificateur, variable libre ou liée, portée d’une variable, table de vérité, …
