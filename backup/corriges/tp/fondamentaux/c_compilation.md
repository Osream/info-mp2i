# Fondamentaux du TP : Compilation en C

* Expliquer les grandes étapes du processus de compilation (préprocesseur, compilation, édition de liens), et connaître les différents types de fichiers existants (code source `.c`, fichier d'entête `.h`, fichier objet `.o`, fichier exécutable).
* Effectuer une compilation simple d'un fichier de code source `.c` avec `gcc` (et l'option `-o` pour renommer l'exécutable), puis exécuter le fichier produit.
* Effectuer une compilation séparée (création des fichiers objets avec l'option `-c` puis création du fichier exécutable).
* Comprendre un message d'avertissement ou d'erreur affiché par `gcc`.
* Utiliser la directive `#include` pour inclure une librairie existante ou un fichier d'entête.
* Créer un fichier d'entête (`.h`) contenant les directives `#ifndef, #define, #endif`.
* Déclarer une fonction dans un fichier d'entête.
* Définir une fonction dans un fichier de code source.
* Écrire la fonction principale `int main() { ... }`, avec ou sans `return`.
* Appeler une fonction dans le `main`.
* Utiliser la fonction `printf` pour réaliser un affichage, avec les formatages `%d, %f, %c, %s`.
* Respecter les bonnes pratiques de programmation.
